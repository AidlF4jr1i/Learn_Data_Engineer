# Python Control Flow & Error Handling

## Deskripsi Singkat
**Control Flow** adalah mekanisme yang mengatur jalannya programâ€”menentukan **instruksi apa yang dijalankan**, **kapan dijalankan**, dan **kapan berhenti**.  
Python menyediakan berbagai konstruksi untuk mengontrol alur eksekusi program, seperti:
- Mengeksekusi kode berdasarkan kondisi tertentu (**percabangan**).
- Mengulang eksekusi kode (**perulangan**).
- Menghentikan atau melewati iterasi tertentu (**kontrol perulangan**).
- Menangani kesalahan agar program tetap berjalan (**error handling**).

Alur pembahasan:
1. **Operasi Boolean** (Comparison, Equality, Logical Operators).
2. **Percabangan** (`if`, `elif`, `else`, ternary operator).
3. **Perulangan** (`for`, `while`, nested loop).
4. **Kontrol Perulangan** (`break`, `continue`, `else` pada loop, `pass`, list comprehension).
5. **Error Handling** (`try-except`, `else`, `finally`, `raise`).

## 1. Operasi Boolean

Sebelum masuk ke percabangan, kita perlu memahami **tipe data Boolean** dan **operator** yang digunakan untuk mengevaluasi kondisi (`True` atau `False`).

Python memiliki dua kelompok operator Boolean:
- **Comparison Operators**: Membandingkan nilai.
- **Equality Operators**: Mengecek kesamaan/ketidaksamaan.
- **Logical Operators**: Menggabungkan ekspresi boolean (`and`, `or`, `not`).

---

### 1.1 Comparison Operators
Digunakan untuk membandingkan dua nilai.

| Operator | Arti                           | Contoh Penggunaan |
|----------|--------------------------------|-------------------|
| `<`      | Lebih kecil dari               | `a < b`           |
| `>`      | Lebih besar dari                | `a > b`           |
| `<=`     | Lebih kecil atau sama dengan   | `a <= b`          |
| `>=`     | Lebih besar atau sama dengan   | `a >= b`          |

**Contoh:**
```python
motor_alex = 3
motor_raihan = 2

print(motor_alex > motor_raihan)  # True
```

---

### 1.2 Equality Operators
Digunakan untuk memeriksa kesamaan atau ketidaksamaan nilai.

| Operator | Arti                | Contoh Penggunaan |
|----------|---------------------|-------------------|
| `==`     | Sama dengan          | `a == b`          |
| `!=`     | Tidak sama dengan    | `a != b`          |

**Contoh:**
```python
nama_tercantum = "Johan"
nama_penerima = "Johan"

# Cara 1
print(nama_tercantum == nama_penerima)  # True

# Cara 2 (langsung di Jupyter Notebook, output: True)
nama_tercantum == nama_penerima
```

---

### 1.3 Logical Operators
Digunakan untuk menggabungkan dua atau lebih ekspresi boolean.

#### `and`
Menghasilkan `True` jika **semua** kondisi bernilai `True`.

**Truth Table:**

| A     | B     | A and B |
|-------|-------|---------|
| True  | True  | True    |
| True  | False | False   |
| False | True  | False   |
| False | False | False   |

**Contoh:**
```python
print(1 < 2 and 10 > 5)  # True
print(1 < 2 and 10 < 5)  # False
```

---

#### `or`
Menghasilkan `True` jika **minimal satu** kondisi bernilai `True`.

**Truth Table:**

| A     | B     | A or B  |
|-------|-------|---------|
| True  | True  | True    |
| True  | False | True    |
| False | True  | True    |
| False | False | False   |

**Contoh:**
```python
print(1 < 2 or 10 > 5)  # True
print(1 > 2 or 10 > 5)  # True
print(1 > 2 or 10 < 5)  # False
```

---

#### `not`
Membalik nilai boolean: `True` menjadi `False`, dan sebaliknya.

**Karakteristik:**
- Negasi dari nilai sebelumnya.
- Biasanya digunakan bersama `and` atau `or`.

**Contoh:**
```python
print(not True)                  # False
print(True and not True)         # False
print(not False and True)        # True

angka_1 = 4
print(not angka_1 > 6 or angka_1 != 9)  # True
```

## 2. Percabangan (Branching)

Percabangan digunakan untuk **mengatur alur program** berdasarkan kondisi tertentu. Python menyediakan:
- `if`
- `elif`
- `else`

**Contoh:**
```python
nilai = 85

if nilai >= 90:
    print("A")
elif nilai >= 80:
    print("B")
else:
    print("C")
```

---

## 3. Perulangan (Looping)

### 3.1 For Loop
Digunakan untuk mengiterasi **iterable** (list, tuple, string, range, dll.).

**Contoh:**
```python
for i in range(5):
    print(i)
```

---

### 3.2 While Loop
Akan terus berjalan selama kondisi bernilai `True`.

**Contoh:**
```python
count = 0
while count < 5:
    print(count)
    count += 1
```

---

### 3.3 Nested Loop
Loop di dalam loop.

**Contoh:**
```python
for i in range(3):
    for j in range(2):
        print(f"i={i}, j={j}")
```

---

## 4. Kontrol Perulangan

### 4.1 break
Menghentikan loop sepenuhnya.

```python
for i in range(5):
    if i == 3:
        break
    print(i)
```

---

### 4.2 continue
Melewati iterasi saat ini dan lanjut ke iterasi berikutnya.

```python
for i in range(5):
    if i == 2:
        continue
    print(i)
```

---

### 4.3 else pada loop
Dijalankan jika loop selesai tanpa `break`.

```python
for i in range(5):
    print(i)
else:
    print("Loop selesai tanpa break")
```

---

### 4.4 pass
Placeholder agar blok kode valid tanpa menjalankan apa pun.

```python
for i in range(5):
    pass
```

---

### 4.5 List Comprehension
Cara singkat membuat list dari loop.

```python
angka = [x**2 for x in range(5)]
print(angka)  # [0, 1, 4, 9, 16]
```

## 5. Error Handling

### 5.1 try-except
Menangkap error agar program tidak berhenti.

```python
try:
    angka = int("abc")
except ValueError:
    print("Terjadi error konversi angka")
```

---

### 5.2 else
Dijalankan jika tidak ada error.

```python
try:
    angka = int("123")
except ValueError:
    print("Error")
else:
    print("Berhasil:", angka)
```

---

### 5.3 finally
Selalu dijalankan, terlepas ada error atau tidak.

```python
try:
    angka = int("123")
finally:
    print("Selesai mencoba")
```

---

### 5.4 raise
Melempar error secara manual.

```python
x = -1
if x < 0:
    raise ValueError("Angka tidak boleh negatif")
```
