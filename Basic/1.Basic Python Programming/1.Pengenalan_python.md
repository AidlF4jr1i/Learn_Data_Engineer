## Printing

`Printing` adalah cara untuk menampilkan output dari suatu perintah ke layar (console).  
Dalam Python, perintah yang digunakan adalah `print()`.

Contoh dasar:
```python
print("Hello World!")    # Menampilkan teks "Hello World!"
print(12)                # Menampilkan angka 12
```

Contoh penggunaan variabel:
```python
nama = "Budi"
print(nama)
```

```python
text = "Saya sedang belajar bahasa pemrograman Python"
print(text)
```

## Variables

Variabel adalah wadah untuk menyimpan data. Data ini bisa berupa angka, teks (string), list, dan lain-lain.

Contoh:
```python
nama = "Ahmad"
no_kelas = 203
buah = ["Semangka", "Lemon", "Apel"]
```

Catatan Penting:
1. Nama variabel tidak boleh diawali dengan angka atau menggunakan simbol unik. Contoh yang salah:
   - `5ama = ...`
   - `#ahh = ...`
   - `Sa$i = ...`
2. Gunakan nama variabel yang deskriptif dan mudah dimengerti.  
   Contoh yang tidak disarankan:
   ```python
   put = "12 Januari 2021"  # Kurang jelas maksudnya
   ```

## Commenting

Komentar digunakan untuk memberikan penjelasan pada kode. Ini penting untuk membantu orang lain (dan diri sendiri) memahami maksud dari kode yang ditulis.

### Jenis komentar:
| Jenis        | Syntax | Keterangan                                                  |
|--------------|--------|-------------------------------------------------------------|
| Single Line  | `#`    | Menjelaskan satu baris kode                                |
| Inline       | `#`    | Komentar di akhir baris kode                                |
| Multi Line   | `"""`  | Menjelaskan penjelasan panjang, biasanya lebih dari 1 baris |

### Contoh:
#### Single Line
```python
# Menghitung profit dari selisih revenue dan modal
profit = revenue - modal
print(profit)
```

#### Inline Comment
```python
sisi_persegi = 5               # Menyimpan nilai sisi persegi
luas_persegi = sisi_persegi**2 # Menghitung luas persegi
```

#### Multi Line Comment
```python
"""
Komentar multi-line biasanya digunakan untuk penjelasan panjang.
Ditandai dengan tiga tanda kutip di awal dan akhir.
Bisa menggunakan kutip tunggal atau ganda.
"""
```

## Error dan Jenis-Jenisnya

Error adalah kondisi di mana program tidak berjalan sebagaimana mestinya karena adanya kesalahan penulisan atau logika.

### Tiga jenis error yang umum:
- Syntax Error
- Indentation Error
- NameError

### Bonus:
- Logical Error

---

### Syntax Error

Terjadi karena penulisan sintaks Python yang tidak sesuai aturan.

Contoh salah:
```python
print("Hello World!"):
```

Output error:
```
SyntaxError: invalid syntax
```

---

### Indentation Error

Terjadi ketika penulisan indentasi (jarak/spasi/tab) tidak sesuai aturan. Python menggunakan indentasi untuk menunjukkan blok kode.

Contoh salah:
```python
def hello()
pass  # ❌ tidak ada indentasi di dalam blok fungsi
```

Contoh benar:
```python
def hello():
    pass  # ✅ menggunakan indentasi yang benar
```

---

### NameError

Kesalahan ini terjadi jika kita memanggil nama variabel atau fungsi yang tidak dikenal oleh Python.

Contoh:
```python
prin("Hello!")  # ❌ Salah ketik fungsi 'print'
```

Contoh lain:
```python
PRINT("Hello")  # ❌ Python bersifat case-sensitive, harus 'print'
```

---

### Logical Error

Error ini terjadi karena kesalahan logika, bukan sintaks. Program tetap berjalan, tapi hasilnya salah.

Contoh:
```python
bagi = 10
pembagi = 0

print(bagi / pembagi)  # ZeroDivisionError: division by zero
```

Untuk menghindari crash saat error terjadi, kita bisa menggunakan blok `try-except`:

```python
try:
    bagi = 10
    pembagi = 0
    print(bagi / pembagi)
except:
    print(f"Oops, terjadi error {ZeroDivisionError} nih")
print("Ikuti perintah ini.")
```

### Best Practice

Gunakan pengecekan error lebih spesifik:

```python
try:
    bagi = 10
    pembagi = 0
    print(bagi / pembagi)
except Exception as e:
    print(f"Oops, terjadi error nih ", e.__class__)
print("Ikuti perintah ini.")
```

```python
try:
    bagi = 10
    pembagi = '0'
    print(bagi / pembagi)
except ZeroDivisionError:
    print(f"Oops, terjadi error ZeroDivisionError nih")
except ValueError:
    print(f"Oops, terjadi error ValueError nih")
except Exception as e:
    print('Error tak dikenal', e.__class__)
print("Ikuti perintah ini.")
```

Note: Bagian `try` dan `except` akan dibahas lebih mendalam di section **"Python for Software Engineering"**, jadi sementara cukup dikenali dulu ya!
