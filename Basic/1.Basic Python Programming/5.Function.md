# 📌 Pengenalan Function (Fungsi) di Python

## Ringkasan
Dalam pengembangan program Python, semakin kompleks sebuah program, semakin tinggi pula risiko terjadinya **duplikasi kode**, **kerumitan pemeliharaan**, dan **potensi bug**. Salah satu solusi untuk membuat kode **lebih ringkas, terstruktur, dan mudah dikelola** adalah dengan menggunakan **Function**.

**Function** adalah blok kode terorganisir yang dapat digunakan kembali (**reusable**) untuk menjalankan tugas tertentu.  
Keuntungan utama menggunakan function:
- ✅ Menghindari pengulangan kode (prinsip **DRY - Don't Repeat Yourself**)
- ✅ Mempercepat pengembangan program
- ✅ Memudahkan debug dan pemeliharaan
- ✅ Membuat kode lebih terstruktur

---

## 1. Mengapa Function Dibutuhkan?

Bayangkan kita ingin menampilkan hasil perkalian beberapa angka.  
Tanpa fungsi, kode bisa terlihat seperti ini:

```python
# Contoh kode kurang efisien

# Menampilkan hasil perkalian 2 dan 3
num_1 = 2
num_2 = 3
print(f"hasil perkalian {num_1} dan {num_2} = {num_1 * num_2}")

# Menampilkan hasil perkalian 2 dan 4
num_1 = 2
num_2 = 4
print(f"hasil perkalian {num_1} dan {num_2} = {num_1 * num_2}")

# Menampilkan hasil perkalian 2 dan 5
num_1 = 2
num_2 = 5
print(f"hasil perkalian {num_1} dan {num_2} = {num_1 * num_2}")

# Menampilkan hasil perkalian 2 dan 6
num_1 = 2
num_2 = 6
print(f"hasil perkalian {num_1} dan {num_2} = {num_1 * num_2}")
```

❌ **Masalah:** Kode diulang-ulang, sulit dikelola, dan memakan banyak baris.

---

## 2. Menggunakan Function

Dengan function, kode menjadi lebih ringkas:

```python
# Membuat fungsi perkalian
def perkalian(num_1, num_2):
    print(f"hasil perkalian {num_1} dan {num_2} = {num_1 * num_2}")

# Memanggil fungsi
perkalian(2, 3)
perkalian(2, 4)
perkalian(2, 5)
perkalian(2, 6)
perkalian(-3, 1.5)
```

**Keuntungan:**
- Hanya menulis logika sekali, lalu **dipanggil berkali-kali**.
- Mudah diubah, cukup edit di satu tempat.

## 3. Konsep Dasar Function

### Sintaks Dasar
```python
def nama_fungsi(parameter_opsional):
    # isi fungsi
    perintah
```

**Keterangan:**
- `def` → keyword untuk mendefinisikan function.
- `nama_fungsi` → identitas fungsi, mengikuti aturan penamaan variabel.
- `parameter_opsional` → input ke fungsi (boleh kosong atau diisi sesuai kebutuhan).
- `:` → mengakhiri deklarasi fungsi.
- **Indentasi** wajib pada isi fungsi.

**Contoh sederhana:**
```python
def belajar_fungsi():
    print("Hello World!")

# Pemanggilan fungsi
belajar_fungsi()
```

## 4. Parameter dan Argumen

Parameter adalah variabel yang didefinisikan pada fungsi, sedangkan argumen adalah nilai yang diberikan saat fungsi dipanggil.

**Contoh:**
```python
def identitas(nama):
    print(nama)

identitas('Rangga')
```

**Multiple Parameters:**
```python
def identitas(nama, asal, umur):
    print(f"{nama} berasal dari {asal} berumur {umur} tahun")

identitas('Rangga', 'Surabaya', 22)
identitas('Fernando', 'Jakarta', 21)
```

⚠️ **Perlu diingat:** Jumlah argumen harus sesuai dengan jumlah parameter.

---

## 5. Percabangan dalam Function

Fungsi bisa mengandung logika **if-else**, **looping**, dan **struktur data**.

**Contoh:**
```python
# Fungsi diskon
def diskon(promo, harga):
    if promo:
        potongan = 50 / 100
        total_harga = int(harga * potongan)
        print(total_harga)
    else:
        print(harga)

diskon(True, 100000)
diskon(False, 100000)
```

## 7. Keyword Arguments

Dengan **keyword arguments**, kita bisa memanggil fungsi tanpa memperhatikan urutan parameter:

```python
def pegawai(nama, departemen, tahun):
    print(f"{nama}, merupakan karyawan dari departemen {departemen} sejak tahun {tahun}.")

pegawai(nama="Chaidar Abdul", departemen="Public Relation", tahun=2010)
pegawai(departemen="Marketing", tahun=2014, nama="Fajariyah Ainun")
```

## 8. Default Parameter Value

Parameter bisa memiliki **nilai default**:
```python
def luas_lingkaran(jari_jari, PI=3.14):
    print(f"Luas lingkaran = {PI * jari_jari * jari_jari}")

luas_lingkaran(4)
luas_lingkaran(8, 3.1416)  # Override default value
```

## 9. Return vs Print

| Return | Print |
|--------|-------|
| Mengembalikan nilai | Menampilkan output |
| Hanya di dalam fungsi | Bisa di mana saja |
| Menyimpan nilai untuk digunakan kembali | Tidak menyimpan nilai |

**Contoh Return:**
```python
def omzet(jumlah_produk, harga_jual):
    return jumlah_produk * harga_jual

final_omzet = omzet(2, 50000)
print(final_omzet)  # Nilai bisa digunakan kembali
```

**Contoh Print:**
```python
def omzet(jumlah_produk, harga_jual):
    print(jumlah_produk * harga_jual)

final_omzet = omzet(2, 50000)  # final_omzet akan bernilai None
```

## 10. *args dan **kwargs

**`*args`** → Menampung argumen dalam bentuk tuple.  
**`**kwargs`** → Menampung argumen dalam bentuk dictionary.

**Contoh `*args`:**
```python
def pelanggan(*nama):
    for n in nama:
        print("Pelanggan:", n)

pelanggan("Farhan", "Fitri", "Chaidar")
```

**Contoh `**kwargs`:**
```python
def pelanggan(**data):
    for key, value in data.items():
        print(f"{key} : {value}")

pelanggan(nama="Farhan", pembelian="Beras")
```

📚 **Kesimpulan:**
Function adalah elemen penting dalam pemrograman Python untuk menciptakan **kode yang efisien, terstruktur, dan mudah dikelola**. Dengan memahami parameter, return value, serta `*args` dan `**kwargs`, kita dapat membangun program yang **fleksibel dan scalable**.
